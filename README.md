## 🖥️ 프로젝트 소개
지능형 분리수거함을 포함하여 사용자별 화장품 마일리지를 적립하여 사용자가 스마트폰으로 언제든지 분리 배출 목록과 마일리지를 확인할 수 있는 선순환 구조의 IoT 시스템 입니다.
<br>
<br>
